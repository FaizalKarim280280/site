Projects

RoadMent

RoadMent is an end-to-end platform for segmentation of roads from satellite images. This is a project based on segmenting out roads from satellite images. We have used the U-Net architecture for the model, coupled with dice coefficient loss for loss function and IoU score and dice coefficient as metrics.
After model training, we got a training IoU score of 0.497 and a validation IoU score of 0.415. Finally, the model was integrated on a website using the Flask API.

Clean/Dirty Road Classification

This is a project based on ConvNets used to identify whether a road is clean or dirty. We have used MobileNet as our base architecture and the weights are based on imagenet. We have used web scraping techniques to download 238 images of both clean and dirty roads.
We achieved a training accuracy of 97.1% and validation accuracy of 91.6%.
